Work

Automated Stock Market Data Scraper

A Python-based system for automating large-scale stock market data extraction from TradingView. It collects and structures detailed information about thousands of U.S. stocks, including prices, performance trends, dividends, profitability ratios, and complete financial statements — all without relying on Selenium or browser automation.

The scraper communicates directly with TradingView’s backend API to fetch real-time data, validate available financial fields, and export structured CSV files ready for financial analysis, visualization, and machine learning workflows.

Built with: Python, Requests, BeautifulSoup, Pandas
Outputs: Clean CSV datasets for performance, profitability, dividends, and financial statements
Applications: Quantitative finance research and automated stock screening

EHL Film-Thickness Prediction

This project focuses on developing machine-learning models to predict elastohydrodynamic lubrication (EHL) film thickness under various operating conditions. It involves preprocessing large experimental datasets, extracting physical and statistical features, and training predictive models using both classical regression and deep learning approaches.

The workflow integrates automated data validation, visualization, and performance evaluation to ensure robust and interpretable model results. The final system provides accurate predictions of film thickness for different material and load configurations, supporting research in lubrication and tribology.

Built with: Python, scikit-learn, TensorFlow, Pandas, NumPy
Tasks: Data preprocessing, feature engineering, model training, and evaluation
Applications: Predictive modeling of EHL film thickness and data-driven analysis of lubrication systems
